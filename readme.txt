Кредитен калкулатор за УниКредит — WooCommerce модул, комуникиращ с Контролния панел и SmartUCF.

== Description ==

Модулът позволява на клиентите да изчислят и изберат кредитен план при покупка на изплащане чрез УниКредит. Настройките и параметрите на кредита се получават от Контролния панел; финалната заявка се изпраща към банковия SmartUCF.

== Installation ==

1. Качете папката `mtunicredit` в `/wp-content/plugins/`.
2. Активирайте плъгина от меню Плъгини в WordPress.
3. Уверете се, че WooCommerce е активен.
4. Конфигурирайте връзката с Контролния панел от настройките на модула.
